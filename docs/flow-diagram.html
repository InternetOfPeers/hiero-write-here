üéØ How It Works in a Nutshell

1. Recipient sets up a message box (creates topic + publishes public key)
2. Sender looks up recipient's topic ID from account memo
3. Sender encrypts message with hybrid encryption and submits to topic
4. Recipient polls topic for new messages and decrypts them
5. All messages are stored immutably on Hedera's blockchain
