## Summary

This system provides **secure, asynchronous messaging** on the Hedera network:

1. **Setup**: Generate keys, create topic, publish public key
2. **Send**: Lookup recipient, encrypt message, submit to topic
3. **Receive**: Poll topic, decrypt messages, display

**Security**: Hybrid encryption (AES-256 + RSA-2048) ensures only the recipient can read messages.

**Simplicity**: Everything is automated - users just run npm commands.

**Scalability**: Supports messages of any size with automatic chunking/reassembly.
